## How to install
1. Decompress the downloaded ``.tar`` into ``/boot/grub/themes/``
2. Add ``GRUB_THEME="/boot/grub/themes/{theme-name}/theme.txt"`` in the GRUB config file in ``/etc/default/grub``
3. Type ``sudo update-grub``
4. Enjoy
